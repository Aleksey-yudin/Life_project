# Детализация функционала проекта

## 1. Авторизация

- Возможность входа на сайт только имея пароль (стартовая страница Login).
- Защита всех личных данных и разделов до прохождения авторизации.
- В будущем — поддержка многопользовательского входа с e-mail + password (опционально).

---

## 2. Дашборд

### На дашборде должны отображаться:
- **График ежемесячных трат** за текущий год (bar- или line-chart).
- **Топ 3 категории** по тратам за последний месяц (текстовый или donut/pie chart).
- **Текущая сумма денег на всех кошельках** (отдельный виджет, общая сумма).
- **Список задач на сегодня** с возможностью отметить выполнение.
- **Напоминания о привычках** — какие привычки сегодня не были отмечены как «выполненные» или «выполнено частично» (список или виджет).
- Меню для быстрого перехода на главные разделы: **«Финансы»**, **«Привычки»**, **«Задачи»** (меню сбоку/сверху).

---

## 3. Финансы

### Верхняя часть страницы:
- **Аналитика по тратам за месяц**: общая сумма + разбивка по каждой категории (график/таблица).
- **Отображение балансов по кошелькам** — перечень кошельков с актуальными остатками.

### Функционал управления:
- Возможность **редактировать, добавлять, удалять** категории расходов и доходов (CRUD-категорий).
- Возможность **редактировать, добавлять, удалять** кошельки: изменять их название и текущий баланс.
- Форма для **добавления трат/доходов** — выбор категории, сумма, дата, выбор кошелька.
- Просмотр всей истории операций с поиском и фильтрацией по дате/категории/кошельку.
- Возможность быстро внести корректировку остатков кошелька (ручное редактирование).

---

## 4. Привычки

### Основные функции:
- **Добавление новых привычек** — возможность задавать название (например, «Чтение», «Зарядка») и цвет/иконку привычки.
- Для каждой привычки — **календарь текущего месяца** с возможностью отметить:
    - выполнено (галочка)
    - выполнено частично (полоса/кружок)
    - пропущено (крест/прочерк)
- **Автоматическая архивация** календарей прошлых месяцев (просмотр архива доступен).
- **Общий календарь настроения**: на каждый день можно выбрать одно из состояний (**хорошее, плохое, стресс, радость**) и добавить текстовый комментарий.
- Просмотр статистики по привычкам (процент выполнения текущего месяца, streak и т.п., виджеты, мини-графики).
- Напоминания о привычках, которые ещё не отмечены — отображать на дашборде.

---

## 5. Задачи

### Календарь и задачи:
- **Краткая сводка задач на текущий день** (отдельный блок/список).
- Интерактивный **календарь** с возможностью переключения между дневным/недельным/месячным режимом.
- **Добавление задач** с заполнением полей:
    - Заголовок задачи (обязательное поле)
    - Описание задачи (необязательно)
    - Точная дата или диапазон дат (задача может длиться несколько дней)
    - Пометка важности (обычная, важная, срочная и т.д.)
- Возможность помечать задачу как выполненную/отложенную/удалённую (или архивировать).
- Просмотр задач по выбранному интервалу в календаре.
- Группировка задач по дням, приоритету или категории (опционально).
- Возможность создавать подзадачи (checklist внутри задачи, по желанию).
