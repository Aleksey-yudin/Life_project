## Что нужно сделать

### 1. Установить зависимости

```bash
cd frontend
npm install
```

### 2. Настроить Supabase

1. Создайте проект в [Supabase](https://supabase.com/)
2. Выполните SQL-скрипт из `SUPABASE_SETUP.md`
3. Настройте RLS политики
4. Включите Email Auth

### 3. Настроить переменные окружения

Убедитесь, что в `frontend/.env.local` указаны правильные ключи:

```env
NEXT_PUBLIC_SUPABASE_URL=https://zwfklojxdghhbeaqcyww.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=sb_publishable_j5pG83lhPvtFTPHTnhPBZA_v2InaKQn
```

### 4. Запустить приложение

```bash
cd frontend
npm run dev
```

Откройте http://localhost:3000

## Основные функции MVP
